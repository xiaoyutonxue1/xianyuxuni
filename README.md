## 开发说明

### 环境要求
- Node.js >= 16
- React >= 18
- TypeScript >= 4.9

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```
