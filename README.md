## 功能特性

### 基础功能
- 商品分类管理
- 商品模板管理
- 商品库管理
- 完整度检查
- 发货方式设置

### 商品管理
- 支持单规格和多规格商品
- 多种发货方式：
  * 百度网盘链接
  * 百度网盘群链接
  * 百度网盘群口令
  * 夸克网盘链接
  * 夸克网盘群链接
- 商品完整度实时计算
- 默认发货方式设置

## 技术栈

- React 18
- TypeScript
- Ant Design 5
- React Router 6
- Zustand
- Vite
- Tailwind CSS
- PostgreSQL

## 开发环境
- Node.js >= 16
- npm >= 8
- PostgreSQL >= 12

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 代码检查
npm run lint
```

## 项目结构

```
src/
├── components/     # 公共组件
├── pages/         # 页面组件
│   ├── Products/  # 商品管理
│   ├── Orders/    # 订单管理
│   ├── Settings/  # 系统设置
│   └── Stats/     # 数据统计
├── store/         # 状态管理
├── types/         # TypeScript类型定义
├── utils/         # 工具函数
└── router/        # 路由配置
```
